GuildAI is an AI-powered Discord server administration assistant that brings intelligent automation to your community management — with built-in safety controls.

Chat with GuildAI directly from the Guildora Hub to manage members, roles, channels, and messages. Every proposed action requires explicit confirmation before execution, so you stay in control at all times.

## Features

- **Streaming chat interface** for natural-language server administration
- **Action confirmation system** — no destructive action runs without your approval
- **Read-only mode** for safe testing and evaluation
- **Configurable AI provider** — choose between Anthropic (Claude) and OpenAI (GPT)
- **Role-based access** — restrict chat and action permissions by role
- **Rate limiting** — per-user and per-role rate limits to prevent abuse
- **Conversation history** — review past interactions and AI-proposed actions
- **Audit logging** — all executed actions are logged with 90-day retention
- **MCP server support** — extend GuildAI's capabilities with external tool servers
